# KeyLogger
# Webcam Keylogger + Silent Camera Snapper

A stealthy Python keylogger that:
- Logs every keystroke with timestamps  
- Silently captures the victim's face via webcam  
- Automatically emails both the log and photo every 60 seconds using Gmail  

Runs completely hidden when compiled or launched with `pythonw.exe`.
